# Código de Conducta

## Nuestro compromiso

Nos comprometemos a mantener una comunidad abierta, respetuosa y colaborativa para todas las personas.

## Conductas esperadas

- Trato respetuoso y profesional
- Feedback técnico constructivo
- Apertura a distintos puntos de vista
- Comunicación clara y orientada a resolver problemas

## Conductas no aceptadas

- Ataques personales o acoso
- Lenguaje ofensivo o discriminatorio
- Publicar información privada sin permiso
- Comportamientos que afecten negativamente al entorno de colaboración

## Aplicación

Las personas mantenedoras pueden moderar discusiones, rechazar contribuciones o limitar participación cuando sea necesario para proteger la comunidad.

## Reporte

Para reportar incidentes de conducta, abre un issue privado o contacta a las personas mantenedoras por los canales del repositorio.

## Alcance

Este código aplica en el repositorio, issues, pull requests y espacios de discusión relacionados con el proyecto.
